│  ├─demo : Demo Functions
│  ├─dev : 文档版
│  ├─doc : Manual
│  ├─mob : Mobile
│  ├─rest : REST-API
│  ├─... Extend freely
│  └─umc : User Center
├─ximps : Static, 3nd Component
│  ├─static : Static Pic,Icon,Logo,Fonts; 
│  ├─vendor : the 3nd Libs/Component
│  └─vendui : the 3nd js/Front Component
